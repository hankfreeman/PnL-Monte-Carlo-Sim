This app uses historical company data from the PnL like salary expense, overhead and commission income.
It allows the user to select the best fit regression for each of these line items.
The resulting profit equation (system of equations) can be used to generate monte carlo simulations to speculate how the company might become more profitable by an aggressive hiring campaign.

[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://pnl-monte-carlo-sim-2bjqwjuavj6.streamlit.app/)

### How to run it on your own machine

1. Install the requirements

   ```
   $ pip install -r requirements.txt
   ```

2. Run the app

   ```
   $ streamlit run streamlit_app.py
   ```
